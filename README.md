# Protocole chat

## Build
    
    Vous devez vous placer dans le dossier racine du projet.

```bash
make
```
## Execution

    Ces commandes sont à éxecuter dans le dossier src. Vous pouvez y accéder avec
```bash
cd src
```

    Pour lancer le serveur :

```bash
java server.Main 
```

    Pour lancer le client :

```bash
java client.Main
```
